# AI-chatbot
A full-stack AI chatbot built using React.js, Python, Ollama, and Firebase Authentication.
Users can securely sign up, log in, and interact with an AI model through a smooth chat interface.

## Features

Secure user authentication with Firebase

Real-time chat interface powered by AI

Backend API handling with Python

Integration with local LLMs using Ollama

Full-stack frontend and backend integration

## Project Highlights

This project helped me gain hands-on experience in:

Full-stack integration (frontend + backend + AI)

Authentication flow using Firebase

API handling with Python

Running and connecting local LLMs with Ollama

Debugging real-world development issues

## Tech Stack

Frontend: React.js

Backend: Python

AI: Ollama LLM

Authentication & Database: Firebase
